Animal Crossing: What's the Catch?

This project is still in the works, but most of the script is finished. Animal Crossing is a social simulation video game series developed and published by Nintendo. One aspect of this game is the ability to catch fish and bugs, which different species are available at different game times, which matches real life time. This website takes the current local time, and displays all fish and bugs that can currently be caught. I will be updating this website to use Material UI for the front end display.
